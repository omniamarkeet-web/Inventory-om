# Inventory Management System

نظام إدارة المخزون - تطبيق لإدارة المخزون والمبيعات والموردين والعملاء مع ربط Google Sheets

## المميزات الرئيسية

- 📊 إدارة المخزون
- 💰 تسجيل المبيعات
- 👥 إدارة العملاء
- 🏭 إدارة الموردين
- 📈 التقارير والإحصائيات
- 🖨️ طباعة الفواتير
- 📱 ربط Google Sheets

## التكنولوجيا المستخدمة

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js / Python
- Database: Google Sheets API
- Authentication: Google OAuth 2.0
